

# ⚙️📘 Ortak Mühendislik Dersleri README

## 🎯 Amaç
Bu repo, **tüm mühendislik bölümlerinde ortak olan temel dersleri** bir araya getirir.  
- Öğrenciler için kolay erişim sağlar  
- Ortak müfredatı anlamayı kolaylaştırır  
- Çalışma ve kaynak paylaşımı için merkezi bir alan oluşturur

Mühendislik dalları farklı uzmanlık alanlarına ayrılır, ancak **temel bilim ve mühendislik dersleri** ortaktır.
