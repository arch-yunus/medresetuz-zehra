
# ⚡ Elektrik-Elektronik Mühendisliği README

## 🎯 Amaç
Bu README, elektrik-elektronik mühendisliği öğrencileri ve profesyonelleri için temel kavramları, kullanılan teknolojileri, kariyer yollarını ve gelecekteki trendleri derli toplu bir şekilde sunar.  

---

## 🖥️ Genel Tanım
Elektrik-Elektronik Mühendisliği, iki ana alanı birleştirir:  
- **Elektrik Mühendisliği**: Enerji üretimi, iletimi, dağıtımı, güç sistemleri  
- **Elektronik Mühendisliği**: Devreler, mikrodenetleyiciler, haberleşme sistemleri, otomasyon  

💡 Kısacası: *“Enerjiyi yönetmek ve elektroniği tasarlamak.”*
