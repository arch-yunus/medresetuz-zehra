

# 🎨⚙️ Endüstriyel Tasarım Mühendisliği README ⚙️🎨

## 🎯 Amaç
Bu README, endüstriyel tasarım mühendisliğini **cafcaflı bir şekilde** tanıtmak için hazırlandı.  
Hem teknik hem de sanatsal yönleriyle bu disiplinin kapsamını, araçlarını ve geleceğini ortaya koyar.  

---

## 🖥️ Genel Tanım
Endüstriyel Tasarım Mühendisliği, **işlevsellik + estetik + ergonomi** üçlüsünü bir araya getirir.  
- **İşlevsellik**: Ürünlerin teknik olarak çalışması  
- **Estetik**: Görsel cazibe, kullanıcıya hitap eden tasarım  
- **Ergonomi**: İnsan odaklı kullanım kolaylığı  

💡 Kısacası: *“Mühendisliğin matematiğini, sanatın ruhuyla buluşturmak.”*
